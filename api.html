API接口列表 2020-04-18
获取LayuiCDN全版本
接口：https://api.layuicdn.com/GetLayuiVersion 复制 打开

返回JSON格式数据，data为各个组件的数组Array 如：layui layer...等，请求方式GET或POST

其中：版本名称（name）、版本号（v）、更新时间（time）

另外：data中的 time: 1564165660 为实际服务器当前的时间戳

注意：如果数组内 例：name = layui，则为该组件最新版本 可凭此数据进行拼接URL

获取Layui压缩文件包
接口：https://api.layuicdn.com/GetCosLayuiZipListObjects 复制 打开

返回JSON格式数据，data为各个组件的数组Array，请求方式GET或POST

其中：文件名称（Key）、更新时间（LastModified）、MD5值（ETag）、文件大小（Size）、储存类型（StorageClass）、完整链接（Url）

另外：data中的 time: 1564165660 为实际服务器当前的时间戳，count：53 为压缩文件数量

获取Layui第三方组件
接口：https://api.layuicdn.com/GetLayuiExtends 复制 打开

返回JSON格式数据，必须为POST请求，传递分页参数 例：page:1，limit:10

其中：名称（name）、版本（version）、状态（state）、唯一ID（token）、文档（url）

另外：拼接组件地址为：https://www.layuicdn.com/{name}/{version}/

获取Layui赞助捐赠列表
接口：https://api.layuicdn.com/GetLayuiSponsors 复制 打开

返回JSON格式数据，必须为POST请求，传递分页参数 例：page:1，limit:10

其中：赞助者（name）、金额（price）、时间（time）、赞助方式（type）

另外：data中的 赞助方式（type），type= [1则为支付宝，2则为微信，3则为其他渠道]

关于API
LayuiCDN API  接口数据均为  GitHub API  接口数据 进行数据缓存，LayuiCDN API更新频率为24小时每次，Methods支持GET、POST、OPTIONS

主要依托于SCF，对GitHub API进行实时监控，并24小时内更新到LayuiCDN API，为了节省平台的开支压力，则将LayuiCDN API的频限设为100QPS/s的频率
